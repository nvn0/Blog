Nix Package Manager Useful Commands

Alguns comandos utéis para o gestor de pacotes Nix.

nix-env --list-generations -> listar gerações
nix-env --rollback -> mudar para a geração anterior
nix-env --switch-generation {numero da geração} -> trocar de geração
nix-env -u -> atualizar programas
nix-env -u {nome do programa} -> atualiza um programa especifico
nix-env -q -> listar programas instalados
nix-env -qa {nome do pacote} -> pesquisar por pacotes
nix-env -iA {nome do pacote} -> instalar pacotes
nix-env --uninstall {nome do pacote} -> desisntalar pacotes
nix-env -e {nome do pacote} -> apagar pacotes
nix-env --set-flag keep true {nome do pacote} -> manter um programa na versão atual e impedir que atualize
